# Real Rail Transit 模组

## 项目简介

Real Rail Transit 是一个基于 Minecraft Java 1.20.6 Fabric 框架开发的专业轨道交通模拟模组，旨在为玩家提供最接近现实的城市轨道交通体验。

## 核心功能

### 1. 真实列车驾驶系统
- **驾驶室交互功能**：高度拟真的驾驶室交互系统，包含大量可交互的实体按钮、开关、仪表盘和显示屏
- **多模式列车驾驶**：支持 ATO（自动驾驶）、CBTC（基于通信的列车控制）、ATP（列车自动防护）、ATC（列车自动控制）、RM（限制人工驾驶）、手动模式等六种驾驶模式
- **安全制动系统**：三级制动系统（正常制动、快速制动、紧急制动），支持超速检测和闯红灯检测
- **目标速度控制**：支持设置目标速度，自动调整到目标速度

### 2. 信号与道岔系统
- **信号机系统**：完整的信号机系统，支持红、绿、黄、引导等标准显示状态
- **道岔系统**：完整的道岔功能，支持手动控制、自动控制和远程调度控制
- **联锁系统**：完整的信号机与道岔联锁逻辑，确保列车运行安全

### 3. 供电系统
- **第三轨供电**：适用于地下和地面线路
- **接触网供电**：适用于地面和高架线路
- **分区控制**：支持不同区段的供电状态控制

### 4. 调度与运行图系统
- **中央调度系统**：提供直观的可视化操作界面，实时监控全线路运行状态
- **运行图编制**：支持详细的列车运行计划制定，包括车次号、始发站、终点站、停站时间、发车间隔等
- **运行图管理**：支持运行图的保存、加载、删除功能

### 5. 车站设施系统
- **屏蔽门系统**：高架屏蔽门和半高屏蔽门，与列车车门联动，支持钥匙控制
- **乘客服务设施**：信息显示设备、便民设施、票务系统等
- **车站建筑**：栏杆、楼梯、玻璃墙、瓷砖墙、天花板、广告牌等装饰方块

### 6. 轨道系统
- **轨道网络**：自动管理轨道连接关系
- **路径查找**：A*算法实现的最短路径查找
- **沿轨道移动**：列车自动沿轨道移动，自动调整方向

### 7. 音效系统
- 列车运行音效（走行音、撞轨音频、受电弓/第三轨取流音）
- 操作音效（开关门音效、制动音效、驾驶室操作音）
- 提示音效（报警音、到站提示音、信号提示音）

### 8. 客户端渲染
- 列车实体渲染器
- 信号机灯光渲染
- 完整的方块模型和状态文件

### 9. 追加包系统
- **追加包管理器**：支持加载和管理自定义列车追加包
- **MTR 追加包转换**：支持将 MTR 模组的追加包转换为本模组格式
- **BB 模型支持**：支持加载和渲染 Blockbench 模型文件
- **追加包资源管理**：支持追加包中的模型、贴图、音效等资源

### 10. 列车照明系统
- **动态照明控制**：支持列车内部和外部照明的动态控制
- **照明模式切换**：支持多种照明模式，模拟真实列车的照明系统

### 11. 调试与运行支持
- **模组运行日志系统**：在服务端内存中记录模组运行过程中的关键信息，支持正常（绿色）、警告（黄色）、错误（红色）三级等级
- **关键事件记录**：列车按运行图发车、进路排列/取消、运行图加载失败、批量线路/车站配置完成、售票机购票成功/失败等都会写入运行日志

## 开发环境要求

- **Java**: JDK 21 或更高版本
- **Minecraft**: 1.20.6
- **Fabric Loader**: 0.18.4 或更高版本
- **Fabric API**: 0.100.8+1.20.6 或更高版本

## 构建说明

1. 克隆项目到本地
2. 运行 `./gradlew build`（Linux/Mac）或 `gradlew.bat build`（Windows）
3. 构建产物位于 `build/libs/` 目录

## 项目结构

```
src/main/java/com/real/rail/transit/
├── api/              # API接口
├── addon/            # 追加包系统
├── block/            # 方块类
│   ├── entity/       # 方块实体
│   └── screen/       # 方块界面处理器
├── cab/              # 驾驶室交互
├── entity/           # 实体类
├── item/             # 物品类
├── registry/         # 注册类
├── station/          # 车站设施
├── system/           # 系统管理（9个系统）
├── track/            # 轨道系统
├── sound/            # 音效系统
├── util/             # 工具类
└── world/            # 世界Tick处理

src/client/java/com/real/rail/transit/
├── client/
│   ├── model/        # 模型加载器（BB模型等）
│   ├── renderer/     # 渲染器
│   └── screen/       # GUI界面（13个界面）
```

## 游戏内指令

- **/rrtlog [limit]**：在聊天栏中查看最近 N 条模组运行日志
  - 等级颜色：INFO 为绿色、WARN 为黄色、ERROR 为红色
  - 例如：`/rrtlog`（默认 50 条）、`/rrtlog 100`
- **/rrtaddon**：在聊天栏发送一条可点击的链接，打开 Real Rail Transit 追加包创建网站
