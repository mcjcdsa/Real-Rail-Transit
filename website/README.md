# Real Rail Transit 模组官网

这是 Real Rail Transit 模组的官方网站，主要用于下载模组和查看新闻公告。

## 网站结构

- `index.html` - 首页，展示模组介绍和核心功能
- `download.html` - 下载页面，提供模组下载链接和安装说明
- `news.html` - 新闻列表页面，展示所有新闻公告
- `news-detail.html` - 新闻详情页面，展示单篇新闻的完整内容
- `about.html` - 关于页面，介绍模组和团队信息
- `css/style.css` - 网站样式文件

## 功能特点

1. **响应式设计** - 支持桌面和移动设备
2. **现代化UI** - 简洁美观的用户界面
3. **独立页面** - 每个板块都有独立的页面
4. **易于维护** - 纯静态HTML，无需服务器端支持

## 使用方法

1. 将所有文件上传到 Web 服务器
2. 确保文件结构保持不变
3. 访问 `index.html` 即可查看网站

## 自定义

- 修改 `css/style.css` 可以更改网站样式
- 修改各个 HTML 文件可以更新内容
- 添加新的新闻只需在 `news.html` 中添加新的新闻项，并创建对应的详情页面

## 浏览器支持

- Chrome/Edge (最新版本)
- Firefox (最新版本)
- Safari (最新版本)
- 移动浏览器
