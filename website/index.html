高度还原真实轨道交通系统

基于 Minecraft Java 1.20.6 Fabric 框架开发的专业轨道交通模拟模组，提供完整的列车驾驶、信号联锁、调度管理等核心功能

🚧 模组正在开发中，首个版本预计不早于 2026年12月 发布

了解详情 查看公告

核心功能

🚂

真实列车驾驶系统

支持六种驾驶模式（ATO、CBTC、ATP、ATC、RM、手动），高度拟真的驾驶室交互系统

🚦

信号与道岔系统

完整的信号机系统、道岔功能和联锁逻辑，确保列车运行安全

⚡

供电系统

支持第三轨供电和接触网供电，分区控制不同区段的供电状态

📊

调度与运行图

中央调度系统，支持详细的列车运行计划制定和管理

🏢

车站设施系统

屏蔽门系统、乘客服务设施、车站建筑等完整的车站设施

📦

追加包系统

支持加载和管理自定义列车追加包，支持 MTR 追加包转换和 BB 模型渲染
