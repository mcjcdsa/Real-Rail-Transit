项目简介

Real Rail Transit 是一个基于 Minecraft Java 1.20.6 Fabric 框架开发的专业轨道交通模拟模组，旨在为玩家提供最接近现实的城市轨道交通体验。本模组由 Real Rail Transit 模组制作工作室开发，致力于将现实中的地铁、轻轨、市域铁路等轨道交通系统完整复现到 Minecraft 世界中。

核心特色

  • 真实列车驾驶系统：支持六种驾驶模式（ATO、CBTC、ATP、ATC、RM、手动），高度拟真的驾驶室交互系统
  • 完整的信号联锁系统：信号机、道岔、联锁逻辑，确保列车运行安全
  • 智能化调度管理：中央调度系统，支持详细的运行图编制和管理
  • 丰富的车站设施：屏蔽门、信息显示、票务系统等完整的车站设施
  • 追加包支持：支持自定义追加包，MTR 追加包转换，BB 模型渲染

技术规格

  • Minecraft 版本：1.20.6
  • 运行框架：Fabric
  • Java 版本：JDK 21 或更高
  • Fabric Loader：0.18.4 或更高
  • Fabric API：0.100.8+1.20.6 或更高
