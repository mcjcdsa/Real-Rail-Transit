安装说明

  1. 确保已安装 Minecraft Java Edition 1.20.6
  2. 安装 Fabric Loader 0.18.4 或更高版本
  3. 下载并安装 Fabric API
  4. 下载 Real Rail Transit 模组文件
  5. 将模组文件放入 Minecraft 的 mods 文件夹
  6. 启动游戏，享受真实的轨道交通体验！
